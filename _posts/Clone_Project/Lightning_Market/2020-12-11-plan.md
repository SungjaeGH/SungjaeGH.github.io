- DB

## 1. Spring vs Node.js
- Spring
  - 자바 플랫폼을 위한 오픈소스 애플리케이션 프레임워크
  - 정형화된 패턴 위주로 개발하며, 자유도는 다소 떨어지지만 안정성이 높음

- Node.js
  - Chrome V8 Javascript 엔진으로 빌드된 Javascript 런타임
  - 정형화된 패턴이 존재하지 않고, npm에 있는 모듈을 활용할 수 있음

즉, 자신의 환경에 맞춰 선택하는 것이 중요!
