## Array
- 여러 데이터를 하나의 이름으로 그룹핑해서 관리하기 위한 Data structure
- 명칭
  - value : 값
  - index : 값들을 식별하는 번호 ([0]부터 시작)
  - element : index + value
- 장점
  - 작고 가벼움 (메모리 적게 소모)
  - 단순함
- 단점
  - 크기가 정해져 있음
  - 기능이 없음
